# Inteligencia emocional en el aula

Landing page académica construida con Next.js, TypeScript y Tailwind CSS para una presentación scroll-driven sobre inteligencia emocional en el aula.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

## Estructura

- `app/`: layout global y página principal.
- `components/sections/`: secciones semánticas reutilizables de la landing.
- `components/ui/`: primitivas visuales reutilizables.
- `components/cursor/`: base para cursor personalizado.
- `components/layout/`: fondos y wrappers de layout.
- `lib/`: utilidades y contenido estructurado.
